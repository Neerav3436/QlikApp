This project is for assessment for Qlik and High Tech Genesis as part of interview process. Attached project is Angular 8 solution
for sorting, palindrome, anagram.

Please do a git clone of this repository.
Go to ~home/QlikApp via your terminal.
do > npm install
do > npm audit fix force (In case you see alot of warnings during installing the packages)
then > ng serve
open browser at https://localhost:4200
